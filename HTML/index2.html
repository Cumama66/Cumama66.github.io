Справочник CSS

Селекторы CSS
определяют, к каким элементам мы хотим применить стиль.
top:

определяет расстояние от верхнего края родительского элемента до верхнего края дочернего элемента.
padding:

устанавливает значение полей вокруг содержимого элемента.
margin:

нужен для установки величины отступа от каждого края элемента.

position:

устанавливает способ позиционирования элемента относительно окна браузера или других объектов на веб-странице.
width:

устанавливает ширину блочных или заменяемых элементов.
background:
позволяет установить одновременно до пяти характеристик фона.
color:

определяет цвет текста элемента.
text-align:

определяет горизонтальное выравнивание текста в пределах элемента.
transition:

устанавливает эффект перехода между двумя состояниями элемента.
transition-duration:

задаёт время в секундах или миллисекундах, сколько должна длиться анимация перехода до её завершения.
overflow:

управляет отображением содержания блочного элемента, если оно целиком не помещается и выходит за область заданных размеров.
border-radius:
устанавливает радиус скругления уголков рамки.
animation-name:

устанавливает одну или несколько анимаций, которые применяются к элементу.
animation-duration:

задаёт время в секундах или миллисекундах, сколько должна длиться анимация движения.
animation-delay:

устанавливает время ожидания перед запуском цикла анимации.
animation-fill-mode:

определяет, какие стили должны применяться к элементу, когда анимация не проигрывается. Например, после её завершения или при остановке.
:hover

определяет стиль элемента при наведении на него курсора мыши.
font-family:
устанавливает семейство шрифта, которое будет использоваться для оформления текста содержимого.
font-weight:

устанавливает насыщенность шрифта.
font-size:

определяет размер шрифта элемента.
font-style:

определяет начертание шрифта — обычное, курсивное или наклонное.
border-bottom

позволяет одновременно установить толщину, стиль и цвет границы внизу элемента.
transition-timing-function

устанавливает, насколько быстро должно изменяться значение стилевого свойство для которого применяется эффект перехода.
max-width:
устанавливает максимальную ширину элемента.
list-style:

задает стиль маркера, его положение, а также изображение, которое будет использоваться в качестве маркера.
content:

позволяет вставлять генерируемое содержание в текст веб-страницы, которое первоначально в тексте отсутствует.
display:

определяет, как элемент должен быть показан в документе.
clear:

устанавливает, с какой стороны элемента запрещено его обтекание другими элементами.
float:

определяет, по какой стороне будет выравниваться элемент, при этом остальные элементы будут обтекать его с других сторон.
opacity:

определяет уровень прозрачности элемента веб-страницы.
text-shadow:

добавляет тень к тексту.
outline:

одновременно устанавливающее цвет, стиль и толщину внешней границы на всех четырех сторонах элемента.
outline-offset:

устанавливает расстояние между рамкой, созданной с помощью свойства outline, и краем или границей элемента добавленной через border.
box-shadow:

добавляет тень к элементу.
background-image:
устанавливает фоновое изображение для элемента.
background-position:

задает начальное положение фонового изображения, установленного с помощью свойства background-image.
background-repeat:

определяет, как будет повторяться фоновое изображение, установленное с помощью свойства background-image.
background-size:

масштабирует фоновое изображение согласно заданным размерам.
Вендорные свойства

Эти свойства работают только в определённых браузерах. Префикс -moz- для Firefox, -ms- для Internet Explorer, -webkit- для Chrome и Safari, -o- для Opera.
