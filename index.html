Справочник CSS

Справочник по тегам и атрибутам HTML

Тег:<'!DOCTYРE html>
нужен для указания типа текущего документа.
Тег: <pre>

нужен для определения блока предварительно форматированного текста.
Тег: <html>

является контейнером, который заключает в себе все содержимое веб-страницы.
Атрибуты

это свойства тега, дающие дополнительные возможности форматирования текста.
Атрибут тега <html> lang="en"

это атрибут, который указывает естественный язык контента веб страницы.
Одинарный тег
это тег, который не имеют закрывающего тега.
Двойной(закрывающий) тег

это контейнеры, парные теги состоят из двух частей — открывающий и закрывающий тег.

Тег: <head>

нужен для хранения других элементов, цель которых — помочь браузеру в работе с данными.
Вложенный тег<meta>

нужен для определения метатегов, которые используются для хранения информации предназначенной для браузеров и поисковых систем.

Вложенный тег <title>

нужен для определения заголовка документа.
Тег<Header>
нужен для задачи «шапки» сайта или раздела, в которой обычно располагается заголовок.
Вложенный тег <H1>

нужен для показания относительной важности секции, расположенной после заголовка.
Тег: <body>

нужен для хранения содержания веб-страницы (контента), отображаемого в окне браузера.

Вложенный тег <P>

нужен для определения текстового абзаца.

Тег: <footer>

нужен для задачи «подвала» сайта или раздела, в нём может располагаться имя автора, дата документа, контактная и правовая информация.
Тег: <link>
нужен для установки связи с внешним документом вроде файла со стилями или со шрифтами.
Тег: <a>

нужен для создания ссылок.
Тег: <ol>

нужен для установки нумерованного списка.
Тег: <li>

нужен для определения отдельного элемента списка.
Тег: <hr>

нужен для создания горизонтальной линии.
Тег: <strong>

нужен для акцентирования текста.
Тег: <ul>

нужен для установки маркированного списка.
Теги: <dl>, <dt>, <dd>,

нужны для создания списка определений.
Тег: <em>

нужен для акцентирования текста.
Тег: <img src=...>

нужен для отображения на веб-странице изображений в графическом формате GIF, JPEG или PNG.
Тег: <br>

нужен для установки перевода строки в том месте, где этот тег находится.
Тег: <table>

служит контейнером для элементов, определяющих содержимое таблицы.
Тег: <tr>

служит контейнером для создания строки таблицы.
Тег: <td>

нужен для создания одной ячейки таблицы.
Тег: <th>

нужен для создания одной ячейки таблицы, которая обозначается как заголовочная.
Тег: <textarea>

нужен для создания области, в которую можно вводить несколько строк текста.
Тег: <thead>

нужен для хранения одной или нескольких строк, которые представлены вверху таблицы.
Тег: <tfoot>

нужен для хранения одной или нескольких строк, которые представлены внизу таблицы.
Тег: <tbody>

нужен для хранения одной или нескольких строк таблицы.
Тег: <i>

нужен для установки курсивного начертания шрифта.
Тег: <div>
нужен для выделения фрагмента документа с целью изменения вида содержимого.
Тег: <span>

нужен для определения строчных элементов документа.
